# **App Name**: Vitalidade Africana

## Core Features:

- Progressive Funnel Engine: An interactive, multi-step quiz experience featuring auto-advance logic and local state persistence to capture user preferences across 40+ dynamic steps.
- Personalized Protocol Generator: A tool that uses reasoning to analyze user health metrics (weight, age, goals) and generate a 'Methodo Africano' report summary tailored to their profile.
- Fluid Visual Navigation: Implementation of Framer Motion for premium slide transitions and micro-interactions that mirror a high-end native health application.
- Real-time Progress Indicator: A persistent top-fixed progress bar that motivates users by visualizing their path toward their personalized diagnostic results.
- Dynamic Social Proof System: Integrated validation screens featuring curated success stories and active community participation metrics to increase trust during the analysis phase.
- Responsive Offer Architecture: A mobile-first checkout landing page with dynamic pricing displays and a customized CTA based on the user's specific pain points.

## Style Guidelines:

- The primary palette features Vibrant Pink (#ff4fa0) for energy and femininity, supported by Success Green (#16c964) for positive feedback loops, all set against a clean off-white (#fafafa) background.
- Recommended pairing of 'Poppins' for elegant, modern headlines and 'Inter' for highly readable body text, ensuring a professional, premium health app aesthetic.
- Use rounded, friendly vector icons with soft gradients to illustrate daily habits, meal types, and body focus areas without looking clinical.
- A mobile-first approach using large cards with soft shadows (#0000000a), centered elements for one-hand operation, and 2-column grid systems for rapid age/habit selection.
- Smooth fade-and-slide transitions between quiz screens to maintain momentum, paired with a celebratory 'confetti' trigger on the final result page.
